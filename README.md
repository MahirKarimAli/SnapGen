# SnapGen AI Video Editor

SnapGen is a web-based, client-side powered, AI-driven video editor designed to bring the capabilities of professional desktop video editors directly to the browser. 

We have prepared a comprehensive Product Requirements Document (PRD) detailing the application's vision, personas, technical architecture, and gstack-aligned dev+ops guidelines.

## Project Documents
*   [Product Requirements Document (PRD)](PRD.md) - Complete functional and non-functional specifications, including system architecture and development guidelines.

## Core Features Under Development
1.  **Multi-track Editor Timeline:** Video, audio, graphics, and subtitle tracks with frame-accurate scrubbing.
2.  **AI Video & Audio Workflows:** Auto-captioning (Whisper), AI text-to-speech voiceovers, and voice noise reduction.
3.  **Local Wasm Export Engine:** Client-side compiling via `FFmpeg.wasm` for maximum scale and zero-egress cost.
4.  **Premium Glassmorphism UI:** Designed with pure modern CSS custom systems for optimal speed and visual aesthetics.
